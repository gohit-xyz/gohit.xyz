## üíª Essential Commands

### File Management

```bash
# Navigate directories
cd /path/to/directory
ls -lah  # List all files with details
pwd      # Print working directory

# File operations
cp source destination       # Copy files
mv source destination       # Move/rename files
rm -rf directory           # Remove directory recursively
mkdir -p path/to/directory # Create nested directories

# Search files
find . -name "*.js"        # Find JavaScript files
grep -r "pattern" .        # Search pattern in files
```

### Process Management

```bash
# View running processes
ps aux | grep node
top       # Interactive process viewer
htop      # Enhanced process viewer

# Process control
kill PID           # Terminate process
killall node       # Kill all node processes
bg                 # Move process to background
fg                 # Bring process to foreground
```

### System Information

```bash
# System stats
uname -a           # System information
df -h              # Disk usage
free -h            # Memory usage
lscpu              # CPU information
```

### Docker Setup


## üîß System Administration

### User Management

```bash
# Create new user
sudo adduser newuser

# Add user to sudo group
sudo usermod -aG sudo newuser

# Switch user
su - newuser

# View user info
id username
groups username
```

### Package Management

```bash
# APT (Debian/Ubuntu)
sudo apt update                    # Update package list
sudo apt upgrade                   # Upgrade packages
sudo apt install package-name      # Install package
sudo apt remove package-name       # Remove package
sudo apt autoremove               # Remove unused packages

# Snap packages
sudo snap install package-name
sudo snap list
sudo snap refresh
```

### Service Management (systemd)

```bash
# Control services
sudo systemctl start nginx
sudo systemctl stop nginx
sudo systemctl restart nginx
sudo systemctl status nginx
sudo systemctl enable nginx   # Start on boot

# View logs
sudo journalctl -u nginx
sudo journalctl -f           # Follow logs
```

### Networking

## üîê Security Best Practices

### SSH Hardening

```bash
# Edit SSH config
sudo nano /etc/ssh/sshd_config

# Recommended settings:
# PermitRootLogin no
# PasswordAuthentication no
# PubkeyAuthentication yes
# Port 2222  # Change default port

# Restart SSH
sudo systemctl restart sshd
```

### Firewall Configuration

```bash
# Configure UFW
sudo ufw default deny incoming
sudo ufw default allow outgoing
sudo ufw allow ssh
sudo ufw allow http
sudo ufw allow https
sudo ufw enable
```

### System Updates

```bash
# Create update script
cat << 'EOF' > ~/update-system.sh
#!/bin/bash
sudo apt update
sudo apt upgrade -y
sudo apt autoremove -y
sudo snap refresh
echo "System updated successfully!"
EOF

chmod +x ~/update-system.sh
```

## üöÄ Advanced Tips

### Bash Scripting Basics

```bash
#!/bin/bash
# backup.sh - Simple backup script

SOURCE="/home/user/projects"
DEST="/backup"
DATE=$(date +%Y%m%d_%H%M%S)

# Create backup
tar -czf "$DEST/backup_$DATE.tar.gz" "$SOURCE"

# Keep only last 7 backups
cd "$DEST"
ls -t backup_*.tar.gz | tail -n +8 | xargs rm -f

echo "Backup completed: backup_$DATE.tar.gz"
```

### Cron Jobs

```bash
# Edit crontab
crontab -e

# Examples:
# Run backup daily at 2 AM
0 2 * * * /home/user/backup.sh

# Update system weekly on Sunday at 3 AM
0 3 * * 0 /home/user/update-system.sh

# List cron jobs
crontab -l
```
